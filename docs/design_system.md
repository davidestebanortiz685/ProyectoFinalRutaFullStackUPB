#Define colores, tipografías y reglas de estilo.
# Sistema de Diseño

## Paleta de Colores
- **Primario (Acción):** #8c29e2 (Indigo vibrante) - Usar en botones principales.
- **Secundario (Soporte):** #13da8a (Esmeralda) - Usar en mensajes de éxito o confirmación.
- **Fondo General:** #1a5ce1 (Gris muy claro) - Color distintivo de la pagina.
- **Texto Principal:** #373e49 (Gris oscuro) - Para legibilidad.
- **Acentos de Error:** #EF4444 (Rojo suave).

## Tipografía
- **Fuente Principal:** 'Times new Roman' o 'Roboto' (Sans-serif).
- **Encabezados (H1, H2):** Bold, color oscuro.
- **Cuerpo:** Regular, tamaño 16px para legibilidad.

## Componentes UI
- **Botones:** Bordes redondeados (border-radius: 8px), padding amplio, sombra suave al hacer hover.
- **Tarjetas (Publicaciones):** Fondo blanco (#FFFFFF), sombra ligera (box-shadow), bordes redondeados.
- **Espaciado:** Usar sistema de grilla de 8px (márgenes de 8px, 16px, 24px).
- **Imagenes:** Imagenes de las comidas
