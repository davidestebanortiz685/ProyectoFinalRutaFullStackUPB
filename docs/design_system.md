#Define colores, tipografías y reglas de estilo.
# Sistema de Diseño

## Paleta de Colores
- **Primario (Acción):** #4F46E5 (Indigo vibrante) - Usar en botones principales y enlaces activos.
- **Secundario (Soporte):** #10B981 (Esmeralda) - Usar en mensajes de éxito o confirmación.
- **Fondo General:** #F3F4F6 (Gris muy claro) - Para evitar la fatiga visual del blanco puro.
- **Texto Principal:** #1F2937 (Gris oscuro) - Para legibilidad.
- **Acentos de Error:** #EF4444 (Rojo suave).

## Tipografía
- **Fuente Principal:** 'Inter' o 'Roboto' (Sans-serif).
- **Encabezados (H1, H2):** Bold, color oscuro.
- **Cuerpo:** Regular, tamaño 16px para legibilidad.

## Componentes UI
- **Botones:** Bordes redondeados (border-radius: 8px), padding amplio, sombra suave al hacer hover.
- **Tarjetas (Cards):** Fondo blanco (#FFFFFF), sombra ligera (box-shadow), bordes redondeados.
- **Espaciado:** Usar sistema de grilla de 8px (márgenes de 8px, 16px, 24px).
